# Gaia Negev
**Neuroscience Researcher | Master's Student at Tel Aviv University**

I am a neuroscience researcher at the Sagol School for Neuroscience, specializing in computational neuroimaging, structural MRI preprocessing and analysis. 
My work focuses on understanding brain structure across generations using large-scale computational pipelines.

---

### Research Interests
*   **High Level Context:** Generational differences in brain structure.
*   **Neuroimaging:** Structural Imaging - T1-weighted MRI Scans.
*   **Methodology:** Big-data computational pipelines, CAT12, SPM, Nilearn, NeuroCombat.
*   **Statistics:** Multiple Linear Regression, GAMM (Generalized Additive Mixed Models).
*   **Teaching:** Python for Neuroscientists.

### Languages & Tools
*   **Languages:** Python, R, Bash, MATLAB.
*   **Neuroscience Software:** SPM, CAT12, FSL, FreeSurfer.
*   **Version Control & Collaboration:** Git, GitHub, GitHub Classroom.
